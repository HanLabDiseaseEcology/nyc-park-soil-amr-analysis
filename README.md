# nyc-park-soil-amr-analysis

Reproducible workflows for analyzing how antimicrobial resistance in NYC park soils varies with environmental and soil chemistry variables

## Notes on data files

Raw and processed data files are not tracked in this repository. This includes CSV files and Excel files. These files are to be downloaded and added locally, in case of raw, or generated, in vase of  processed, and added to the appropriate `data/raw/` or `data/processed/` folders when running the analysis locally.

The repository is set up to ignore common macOS system files, including `.DS_Store`, because this project is developed on a Mac.
